# /check-context

Read and summarize the current state from your context files.

## Steps

1. Read master context file (e.g., `.context/AI-HANDOFF-CONTEXT.md`)
2. Read strategy file (e.g., `memory/STRATEGY-CONTEXT.md`)
3. Summarize:
   - Active projects and their status
   - Client pipeline and leads
   - Revenue status
   - Top priorities for today
   - Any pending decisions or blockers
